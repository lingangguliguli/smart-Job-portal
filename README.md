## ⚙️ How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/lingangguliguli/smart-Job-portal.git
```

2. Navigate to the project directory

```bash
cd smart-Job-portal
```

3. Install dependencies

```bash
npm install
```

4. Run the development server

```bash
npm run dev
```
